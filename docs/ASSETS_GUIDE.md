# Official Assets Guide

This repository should use only real project assets supplied by the project owner.

Do not add generated placeholder images as official screenshots or logos.

## Recommended folders

```text
docs/assets/branding/
docs/assets/screenshots/
docs/assets/prototypes/
docs/assets/diagrams/
```

## README rule

Only reference files in the README after the images are committed to the repository.

This avoids broken links and avoids presenting concept art as an official product screenshot.
